# CJTL-LDS Personal Website

This is a cyberpunk-themed personal portfolio website hosted on GitHub Pages.

## Features

- **Cyberpunk Aesthetics**: Neon colors, dark background, scanlines, and glitch effects.
- **Matrix Rain Background**: Animated background using HTML5 Canvas.
- **Responsive Design**: Works on desktop and mobile devices.
- **Sections**: Home, About, Projects, Contact.

## Customization

1.  **Content**: Edit `index.html` to update your personal information, projects, and contact details.
2.  **Styles**: Edit `style.css` to change colors (defined in `:root` variables) or layout.
3.  **Animation**: Edit `script.js` to modify the matrix rain effect (speed, characters, colors).

## Deployment

Simply push these files to your GitHub repository `CJTL-LDS.github.io`. GitHub Pages should automatically serve the `index.html` file.
